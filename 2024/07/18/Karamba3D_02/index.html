Karamba3D 02-模型單位與材料斷面參數設定

Author: BSC Date: 七月 18, 2024   Category: Karamba3D

模型單位

Karamba3D 在安裝初期可選擇 SI 制單位或是英制單位進行安裝，如當初選擇 SI 制單位，力量單位為 kN，長度為 m，其他特別標示者則以標示為主。

材料參數設定

Karamba3D 提供三種材料參數設定的方法，分別如下

Material Properties

透過自行輸入材料參數建立材料資料，可建立同向性材料(如:鋼)與正交異向性材料(如:木)

Material Selection

藉由 Karamba3D 內建的材料清單選取預計使用的材料

Read Material Table from File

透過 CSV 檔將材料參數讀入做使用

斷面參數設定

Karamba3D 提供三種段面參數設定的方法，分別如下

Cross-section

透過自行輸入斷面參數建立斷面資料

Cross Section Range Selector

透過輸入深度(height)與寬度(width)，在內建的斷面清單中進行斷面選取。

Read Cross Section Table from File

透過 CSV 檔將斷面參數讀入做使用

註:本文使用 Karamba3D 版本為 3.1.4 版
